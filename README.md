Dự án MovieWeb được xây dựng MERN Stack
Frontend: React.js + Redux Toolkit + Axios + React Router + TailwindCss
Backend: Node.js + Express.js + MongoDB + Mongoose
Chức năng chính:
  -Đăng ký, đăng nhập (JWT)
  -Xem list phim
  -Tìm kiếm phim
  -Thêm phim vào danh sách yêu thích
  -Đổi mật khẩu, ảnh đại diện , cập nhật thông tin
Công nghệ sử dụng
  -MongoDB – lưu trữ dữ liệu NoSQL
  -Express.js – xây dựng REST API
  -React.js – giao diện web
  -Node.js – backend server
  -JWT – xác thực người dùng
  -bcryptjs – mã hóa mật khẩu
  -Axios – gọi API từ frontend
Cài đặt dự án
  -git clone https://github.com/TyLeDuc9/MovieWeb.git
  -cd MovieWeb
Cài đặt Backend
  -cd backend
  -npm install
Tạo file .env
  -MONGO_URL=mongodb://localhost:27017/yourdb
  -JWT_SECRET=your_secret_key
Chạy Backend
  -npm start
Cài đặt Frontend
  -cd frontend
  -npm install
Tạo file .env
  -VITE_TMDB_API_KEY=your_key
Chạy Frontend
  -npm run dev
